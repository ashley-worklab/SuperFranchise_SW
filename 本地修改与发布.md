# 本地修改 → 推送到 GitHub → 在线展示

## 1. 本地修改

- **项目目录**：`C:\Users\davidwang\Documents\genspark-web`
- 用 Cursor / VS Code 打开该文件夹，或直接编辑：
  - **页面内容**：`index.html`
  - **样式**：`css/style.css`
  - **交互与图表**：`js/main.js`
- 本地预览：用浏览器直接打开 `index.html`，或右键「用浏览器打开」。

## 2. 推送到 GitHub（二选一）

### 方式 A：用脚本（推荐）

在项目文件夹里双击运行 **`推送.ps1`**，或在终端执行：

```powershell
cd C:\Users\davidwang\Documents\genspark-web
.\推送.ps1
```

按提示输入本次修改说明（或直接回车用默认说明），脚本会自动执行 `git add` → `commit` → `push`。

### 方式 B：手动命令

在项目文件夹打开终端（PowerShell 或 CMD），执行：

```powershell
cd C:\Users\davidwang\Documents\genspark-web
git add .
git commit -m "更新：这里写你改了什么"
git push origin main
```

## 3. 在线展示

- 推送后 GitHub Actions 会自动部署（约 1～2 分钟）。
- 网站地址：**https://davidsea-z.github.io/GameHotelModel/**
- 部署状态：https://github.com/Davidsea-z/GameHotelModel/actions

**流程总结**：本地改 → 运行 `推送.ps1` 或手动 `git add / commit / push` → 等 1～2 分钟 → 打开上面链接查看更新。
